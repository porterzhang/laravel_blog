## 说明，使用laravel做的简单博客系统
    1、学习Laravel的第一个工程
    2、配置文件忽略
    3、在下载后，执行composer up 命令执行完善基本框架
    4、执行npm install
    5、执行php artisan migrate 进行数据库的完善
